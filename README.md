Task Management REST API
A production-grade RESTful API for task management with JWT authentication, built with FastAPI and PostgreSQL.
Features

JWT Authentication: Secure access and refresh tokens with bcrypt password hashing
User Management: User registration and authentication
CRUD Operations: Complete task management (Create, Read, Update, Delete)
Filtering: Filter tasks by status and priority
Statistics: Task analytics and summary endpoints
Database Design: Properly structured PostgreSQL schema with indexes
API Documentation: Auto-generated Swagger/OpenAPI docs
Security: OWASP best practices, password hashing, token expiration
